## 🚀 Deploying to Vercel

### Option 1 — Vercel CLI (fastest)
```bash
# Install Vercel CLI globally if you haven't already
npm i -g vercel

# From the project root
vercel

# For production
vercel --prod
```

### Option 2 — GitHub → Vercel (recommended for ongoing updates)
1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repo — Vercel auto-detects it as a static site
4. Click **Deploy** — it's live in ~30 seconds ✨

### Option 3 — Drag & Drop
1. Zip the folder
2. Go to [vercel.com/new](https://vercel.com/new) → drag & drop the zip

No build settings needed. Output directory is the project root.

## ✏️ Customizing

**To change the contact email** — search for `dubsglr@gmail.com` in `index.html` and `main.js`, replace with whatever you like.

**To adjust popup timing** — edit the `setTimeout(openPopup, 3500)` value in `main.js` (milliseconds).

**To add a new section** — add a `<section class="section">` block in `index.html` and update the nav.

**Color palette** — all colors are CSS variables at the top of `styles.css` under `:root`.
